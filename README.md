# Secret Message Link

Static web app for creating shareable secret-message links with:
- Custom text
- Optional song link (YouTube / Spotify / etc.)
- Contact links (Facebook, Instagram, LINE ID)
- View counter using countapi.xyz (unique per browser/device)

## How to use with GitHub Pages

1. Create a new public repository on GitHub.
2. Upload `index.html` to the root of the repository.
3. Go to **Settings â†’ Pages**.
4. Under **Source**, select `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`, then save.
6. After deployment, GitHub will give you a URL like:

   `https://<username>.github.io/<repository-name>/`

Open that URL to start creating and sharing secret messages.
